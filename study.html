    레벨별로 매일 자동 출제됩니다. 직접 카드를 추가하면 라이브러리에 더해집니다.

    + 카드 추가

    이렇게 쓰세요

    ① 학생을 등록하세요.
    1. 이름·레벨·수강권(3/6개월)·시작일 입력 → 등록
    2. 학생마다 개인 숙제 링크가 자동 생성됩니다

    ② 링크를 딱 한 번만 카톡으로 보내세요.
    1. 학생 목록의 🔗 버튼 → 링크 복사 → 카톡 전송
    2. 학생이 즐겨찾기/홈화면에 추가하면 끝!

    ③ 그 다음은 자동입니다. 🎉
    1. 매일 아침 보낼 필요 없음 — 학생이 링크를 열면 그날 날짜의 숙제가 자동으로 떠 있어요
    2. 학생은 원어민 발음을 듣고 → 녹음 → 점점 길어지는 문장까지 연습
    3. 숙제 완료 시 선생님께 "○○님 숙제 완료" 알림 메일이 자동 발송됩니다
    4. 학생은 녹음 파일을 카톡으로 원탭 전송 → 선생님은 편한 시간에 피드백

    ④ 패턴 카드는 레벨별로 자동 출제됩니다. 본인 커리큘럼을 '패턴 카드' 탭에서 추가하면 그날부터 반영돼요.
